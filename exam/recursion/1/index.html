# 选择排序

原理

简单选择排序使用简单比较方法在无序区中选择最小的元素放置于开头处

# 程序

void selectSort(int a[], int n, int i)
{
	int k = i; //存放当前最小值的下标
	if (i == n - 1) return;
	//i前面的内容已经序列化,所以从i + 1开始
	for (int j = i + 1; j < n; j++) 
	{
		if (a[j] < a[k])
		{
			k = j;
		}
	}

	if (k != i)
	{
		swap(a[i], a[k]);
	}

	selectSort(a, n, ++i);
	
}
